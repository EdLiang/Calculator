Welcome to my Calculator Simulator

How it works:

This calculator works as a normal calculator would.
1. Start off by inputting a number by using the number buttons.
  A. If no number is input and the user tries to use an operation, you will be prompted to press a number first.
2. After inputting your first number, you may now press an operation key.
3. After pressing an operation key, you may:
  A. Press another operation key if you want to change the current operation.
  B. Input the next number.
4. After inputting the next number, you may:
  A. Press the clear key ONCE to input a new second number.
  B. Press the Equal sign to evaluate.
  C. Press another operation key.
5c. If another operation key was pressed, repeat steps 3 and 4.
5b. If the Equal sign was pressed, you may:
  A. Press another operation key if you would like to operate on the current result.
  B. Press the Clear button to restart.
6. If the user clicks the Clear button twice at any point in time, the calculator will be reset.
